# 华科校园跑

## 功能简介

可以通过填写配置文件实现：

- 一键运动
- 一键提交

## 环境需求

- 进入项目根目录 Hust-Run
- pip install -r requirements.txt
- 本地安装雷电模拟器(网上自行搜索：https://www.ldmnq.com/)
- 将仓库内的 apk 文件拖拽到模拟器上安装

## 使用方法

- 初次需要进行登录
- 之后启动华中大体育
- 随后在根目录 git clone https://github.com/aDarkMaker/Hust-Run.git
- cd Hust-Run
- python src/main.py
- 运行完成后，即可打卡今日的校园跑
